INSPIRATION
-----------

The idea is based on a recursive graph algorithm.
Using this algorithm we can easily predict if the object will be moved.
(See ./Examples/ for details)

RUNNING (ARGS ORDER)
--------------------

seed - len_of_grid_side - multiplier
Too many buttons will cause warnings or errors
So does huge multiplier and incorrect arguments

REQUIREMENTS
------------

The following modules are required:

*[FLTK]
(https://www.fltk.org/software.php)
